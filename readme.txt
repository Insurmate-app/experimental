# Folder Information

This README file provides information about the project's main components: "Gorq", "Ollama", and "Ontology".

## Gorq Folder

The "Gorq" folder contains files and resources related to the Groq integration project. This may include source code, documentation, and other project-specific materials.

## Ollama Folder

The "Ollama" folder contains files and resources related to the Ollama project. This may include source code, documentation, and other project-specific materials.

## Ontology Folder

The "Ontology" folder contains semantic validation rules and document classification logic. It includes:
- Document classification rules
- RDF (Resource Description Framework) implementations
- Semantic validation patterns
- Document type definitions and rules
